  Hexo搭建个人博客还是不错的，写的内容可以很方便在永久保存在github.com或coding.net中，再不也用担心写的东东丢失了。

## 安装

### 安装前提

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

#### [brew](http://brew.sh/)
```bash
/usr/bin/ruby -e \
 "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

> 安装
```bash
sudo brew install unrar
```
> 卸载：
```bash
sudo brew uninstall unrar
```
> 搜索：
```bash
sudo brew search /lighttpd*/
```

#### [Node.js](http://nodejs.org/)
```bash
sudo brew install node
```

#### [Git](http://git-scm.com/)
```bash
sudo brew install git
```

### 安装 Hexo

```bash
npm install -g hexo-cli
```

## 建站

```bash
hexo init <folder>
cd <folder>
npm install
```

## 主题

```bash
cd your-hexo-site
git clone https://github.com/iissnan/hexo-theme-next themes/next
```
修改站点配置文件_config.yml：
```bash
theme: next
```

## SEO优化
更改index.swig文件，文件路径是your-hexo-site\themes\next\layout，将下面代码：
```bash
{% block title %} {{ config.title }} {% endblock %}
```
改成
```bash
{% block title %} {{ config.title }} - {{ theme.description }} {% endblock %}
```

## 配置
修改主配置文件_config.yml
```yml

  branch: master
```

## 常用命令

```bash
hexo new(或者n) "postName" #新建文章
hexo new(或者n) page "pageName" #新建页面
hexo new(或者n) page tags #添加标签tags页面
hexo new(或者n) page categories #添加分类页面
hexo generate(或者g) #生成静态页面至public目录
hexo server(或者s) #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
hexo deploy(或者d) #将.deploy目录部署到GitHub
```
